# Doreads API

APi for Doreads APP

## Requirements

- Node 18.x

## Dependencies

- Express JS
- Dotenv
- prisma

## Commands

### Installation

```bash
 npm install
 yarn intall
```

### Run

```bash
npm run build ## compile source
npm run start ## start compiled source on dis/ folder
npm run dev ## watch mode
npm run lint ## Run linter
npm run test ## Run tests

yarn build
yarn start
yarn dev
yarn lint
yarn test
```
